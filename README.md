Sistema de Leilões – Projeto Java (NetBeans)

Este projeto foi desenvolvido como parte da disciplina de Desenvolvimento de Software, utilizando **Java**, **Swing (JFrame)** e **MySQL** para persistência dos dados.

O sistema possui duas telas principais:

- CadastroVIEW – Tela inicial do sistema, responsável pelo cadastro de produtos.
- ListagemVIEW – Tela que exibe todos os produtos cadastrados no banco de dados.


 Funcionalidades Implementadas

 Definição da tela principal
A tela **CadastroVIEW** foi configurada como a primeira tela a ser aberta ao iniciar o sistema.

 Cadastro de produtos
Funcionalidade completa do botão **Salvar**, gravando os dados no banco de dados.

 Mensagens de confirmação
Após o cadastro, mensagens informam se o processo ocorreu com sucesso ou não.

 Listagem de produtos
Exibe todos os itens cadastrados utilizando a ListagemVIEW.


 Tecnologias utilizadas

- **Java 17**
- **Swing / JFrame**
- **MySQL**
- **JDBC**
- **NetBeans IDE**
- **Git e GitHub para versionamento**
