Nome do Projeto:
Sistema de Gerenciamento de Casa de Leilões

Descrição do Projeto:
Este projeto é um sistema desenvolvido para auxiliar uma casa de leilões no gerenciamento de suas rotinas internas.
Ele permite controlar itens, usuários, lances e demais operações relacionadas ao processo de leilão.
O sistema faz parte de uma atividade prática envolvendo versionamento com Git e GitHub, prática de organização de projeto e uso de banco de dados.

Tecnologias Utilizadas:
Java
MySQL
JDBC
Git / GitHub
NetBeans
